# 🔐 API Standards & Security

## 🎯 Objective
Expose robust, secure, and developer-friendly interfaces.

## 🔌 API Design
- **RESTful Principles:** Resource-based URLs, HTTP methods, status codes.
- **Contract-First:** OpenAPI/Swagger documentation.
- **Versioning:** URI versioning or Header-based versioning.

## 🛡️ Security
- **AuthN/AuthZ:** OAuth2, OIDC, JWT with secure rotation.
- **OWASP Top 10:** SQL injection, XSS, broken access control.
- **Encryption:** TLS everywhere, encryption at rest for sensitive data.
- **Validation:** Strict input validation and sanitization.
